# CodeAlpha-TaskAutomation-Task4
## Basic Rule-Based Chatbot

### What the script does
This is a simple command-line chatbot built in Python. It responds to predefined user inputs like greetings, questions about mood, asking for its name, and saying thanks. The bot runs in a loop until the user types `bye`.

### Key Concepts Used
- **if-elif statements**: To match user input and return the correct response
- **Functions**: `get_response()` and `run_chatbot()` to organize code
- **Loops**: `while True` loop to keep the chat running
- **Input/Output**: `input()` to get user messages and `print()` to show bot replies
- **String methods**: `.lower()` and `.strip()` to handle different input formats

### How to Run
1. Make sure you have Python 3 installed
2. Save the file as `chatbot.py`
3. Open terminal/command prompt in the same folder
4. Run the command:5. Type messages like `hello`, `how are you`, `what's your name`, `thanks`, or `bye` to exit
